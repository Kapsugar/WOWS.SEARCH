# 评分算法(挖坑)
## ALL
鼓励抢人头,收人头,高场均击杀比会比高场均伤害更容易获得更好的评分.
## CV DD
鼓励CV做视野,和收人头,善用毛子的防御占比和占领占比的数据,低场均伤害高胜率高点亮伤害一样可以获得高评价.
## CA BB
其实这个地方我也不知道怎么搞,其实CA还好,CA应该更注重胜率,而BB应该更注重潜在伤害,可是潜在伤害毛子的接口数据我没有找到,可能是我星际玩家了.
